My Projects

Collision Detection and Avoidance System : The project focuses on enhancing road safety by developing an automated system that uses sensors like RADAR, LiDAR, and ultrasonic modules to detect obstacles and prevent collisions. It autonomously applies brakes, issues warnings, and integrates features such as adaptive cruise control and lane departure assistance.

Bluetooth Local Area Network : The project focuses on creating a short-range wireless communication system for efficient data sharing among devices within a limited range. Utilizing Bluetooth technology, the network enables seamless connectivity and supports applications like file transfer, device synchronization, and multimedia sharing. This project emphasizes low-power consumption, ease of deployment, and security measures, making it suitable for personal and professional environments.
